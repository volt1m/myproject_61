# 测试用例详情

## 房产管理模块测试用例

### TC-001: 添加房产信息
**前置条件**: 用户已登录系统
**测试步骤**:
1. 进入房产管理页面
2. 点击"添加房产"按钮
3. 填写必填字段（标题、价格、面积）
4. 点击"保存"按钮

**预期结果**:
- 系统提示"添加成功"
- 新房产显示在列表中
- 数据库记录正确插入

### TC-002: 搜索房产
**测试数据**: 价格范围 50万-200万
**测试步骤**:
1. 在搜索框输入关键词"公寓"
2. 设置价格筛选条件
3. 点击"搜索"按钮

**预期结果**:
- 显示价格在范围内的公寓房源
- 搜索结果数量正确
- 分页功能正常

### TC-003: 编辑房产信息
**测试步骤**:
1. 在房产列表点击"编辑"
2. 修改房产价格
3. 点击"更新"按钮

**预期结果**:
- 系统提示"更新成功"
- 列表中显示更新后的价格
- 数据库记录正确更新
